# OVERVIEW
The aim of this project is to register and store the complaints of the common people.
The information in the storage will be;
- Date of the registration
- Name of the user
- Identification no. like Aadhar
- Complaint paragraph
# FEATURES
- The code is designed to perform multiple tasks such as entering , viewing and deleting complaints.
- The design is simple , yet efficient and easy to understand for the users.
- the program to view all complaints is prepared , whereas , the program to delete all complaints is not , due to security reasons.
# TOOLS USED FOR THE PROJECT
- IDLE (python 3.11 64-bit)
- CSV file (imported from python)
# STEPS TO USE
